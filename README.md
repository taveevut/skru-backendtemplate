# Vali-Admin Template
> นำเอา Template Vali-Admin มาแยกส่วนให้เป็น Template ของเราอีกที
## ขั้นตอนการติดตั้ง
- clone project
```sh
$ git clone </path/to/repository>
```

- Reload Package.json
```sh
$ npm install
```
